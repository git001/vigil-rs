vigil + h2o

This page is served by h2o 2.2.x, supervised by vigild.

Useful endpoints

  • GET /healthz — liveness probe (200 OK)
  • GET /status — h2o internal status

Interact with the supervisor

# from inside the container:
vigil --socket /run/vigil/vigild.sock services
vigil --socket /run/vigil/vigild.sock checks
vigil --socket /run/vigil/vigild.sock logs

# restart h2o (e.g. after config change):
vigil --socket /run/vigil/vigild.sock restart h2o
